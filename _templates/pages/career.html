New PhD positions - Visualization and Machine Learning

A position is open for a brilliant PhD candidate to work on developing exiting scientific visualization and machine learning algorithms. The candidate will help build smart state-of-the-art open source applications that visualize brain and nano data.

Skills required: At least one machine learning course at graduate or undergraduate level, deep learning (optional but bonus), OpenGL or VTK, good background in applied mathematics, Python and C programming, TOEFL score ( > 100), excellent GRE scores (GRE writing > 3.0). The candidate will need to have excellent communication skills and be happy to work in teams and develop open source software.

Information about PhD applications in Intelligent Systems Engineering is available here.

Before applying we recommend e-mailing Prof. Eleftherios Garyfallidis with your CV attached.
